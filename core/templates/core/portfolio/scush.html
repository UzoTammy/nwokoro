Scush Web App

The Website

Scush, a customized SaaS app, that streamlines administrative functions within a company, focusing on three core areas: Stock, Customers, and Human Resourcesâ€”aptly reflected in its name, SCusH. Launched in June 2022, this dynamic website has improved operational efficiency. From processing payroll and tracking key performance indicators to documenting policies and automating, Scush has significantly reduced workloads and minimized chaos and conflict in the workplace. Moreover, its capabilities extend to financial analytics, offering insightful analysis of Profit and Loss statements and balance sheets from accounting software. This feature is pivotal in informing critical business decisions. Scush has not only automated but also transformed administrative processes, proving to be an invaluable asset for Ozone F & L (The owner of the site).

Technologies

  • Programming Language: Python programming language and Javascript
  • Database: Sqlite for developement and Postgres for production on heroku
  • Backend: Django framework technology
  • Frontend: Bootstrap CDN, HTML, and CSS
  • Storing Media Files: Hosted on amazon S3 storage
  • Serving Static Files: Whitenoise
  • Email Server: Google gmail
  • Message Broker: RabbitMq a heroku add-on
  • Domain Source: Namecheap
  • Deployment: Deployed on Heroku platform
  • Collaboration: Git and GitHub
  • Icons: Fontawesome

Main Features

  • Sending Mails: Able to send mails automatically upon updating or creating records
  • Creating PDF: Produce a PDF document as a hard copy
  • Able to carryout scheduled tasks: Send notification, mails and execute tasks
  • Present plots and graphs: A pictorial presentation to aid understanding and analysis of data
